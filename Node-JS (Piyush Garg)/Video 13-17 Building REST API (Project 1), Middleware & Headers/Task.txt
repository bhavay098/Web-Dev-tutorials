Building REST API - JSON

GET /users - HTML Docuemnt Render (for browser)
GET /api/users - List all users JSON (for mobile app)
GET /api/users/1 - Get the user with ID 1
GET /api/users/2 - Get the user with ID 2 (Dynamic Path Parameters)

GET /api/users/:id
:id -> variable | Dynamic

POST /api/users - Create new user

PATCH /api/users/1 - Edit the user with ID 1

DELETE /api/users/1 - Delete the user with ID 1
